# QWOP-RL

## Description
Using reinforcement learning to train an agent to play QWOP. 

## Instructions
Install dependencies
```
pip install pipenv
pipenv install
pipenv shell
```

Start HTTP server for the game itself
```
python host_game.py
```

Train agent (optional)


Test / run agent
